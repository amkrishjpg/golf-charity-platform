# 🏌️ Golf Charity Subscription Platform

Full-stack web application combining subscription, golf score tracking, monthly draws, and charity contributions.

---

## 📂 Project Structure
golf-charity-platform/
│
├── client/ # Frontend (React - Vite)
├── server/ # Backend (Node + Express)
├── docs/ # PRD, notes
│
├── .env
├── .gitignore
├── README.md
└── package.json


---

## 🚀 Getting Started

### Backend

cd server
npm install
npm run dev


### Frontend

cd client
npm install
npm run dev
